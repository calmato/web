## System Overview

塾の運営に必要な以下機能を備えたシステム。

- 授業スケジュール・シフト管理機能
- 成績管理機能
- テスト作成機能  

大学3年時に開発・リリースし、現在も運用に携わっている。
中でもテスト作成機能が最も利用されており、生徒の成績向上に役立ったと好評を頂いている。
